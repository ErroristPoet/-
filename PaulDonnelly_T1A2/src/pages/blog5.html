Align-items and iframe woes

November 10th, 2021

I found I had issues with align-items when I set the parent container height to 100%,
and when I changed it to viewport height (100vh), i.e. that the parent container
is exactly the size of the viewable height of the device used, it worked.
Very frustrating, immense pay-off.
Similarly, huge emotions for fixing my travel page when I created a separate
container for title headings, to get them to line up just so with the div for iframe Google Map images!
